Simple Counter in Vue JS
The following is a simple counter that's been created using Vue.JS Instead of using query selectors as we would inside of vanilla Javascript, we can simply apply a script that we bind to the button element using Vue's v-on:click directive property. This enables us to avoid writing much of the logic that Javscript would require us to write and also enables us to build applications that scale more easily in development.
