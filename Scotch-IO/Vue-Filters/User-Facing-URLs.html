User Facing URLs
If you'd like to clean up data before you show it users, but still save the data in it's original format. You can do so using Vue's filter property. In the example below I'm referencing one single piece of data, a URL that has a tracking parameter. But we don't want to show this URL tracking parameter to users, so to clean it up, we've created a regex function and then connected it to our filter which we've named BareURL. This filter removes any characters including and that then follow the question mark sign.
URL without applying a filter

{{ url }}

URL with applying a filter

{{ url | BareURL }}
